LINUX B6

A UNIX és Linux operációs rendszereknél a telepítésekhez használt szoftvercsomagok forrásfájlok tömörített archívumaiból állnak. Egy-egy csomag úgy "épül föl", hogy különböző erőforrásokkal rendelkező célgépeken is futhasson. Egy szoftvercsomagnak tehát különböző megtestesülései lehetnek: futhat egy Intel dobozban, egy DEC Alpha-ban, egy RISC munkaállomáson, vagy akár egy mainframe-ben. A telepítés felelőssége pedig a végfelhasználóra száll, a billentyűzetnél ülő "rendszergazdára", azaz Rád. Csak semmi pánik! Az egész nem is olyan szörnyű és titokzatos, mint amilyennek gondolnád. Erről a  Következőkben bemutetott példák győzhetnek meg.

  • Linux Mint 19 Cinnamon telepítése Oracle VM 5.2 Virtualboxban
  • Adatmegosztás az Oracle VM 5.2 Virtualboxban
  • Eclipse IDE telepítése
  • A Linux Mint főmenü szerkesztése
  • Az általános nyomtatványkitöltő telepítése
